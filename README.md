# Laravel Broadcast

基于 Go 实现的 `laravel-echo-server` 。在我司内部实现 `notification` 的时候，发现 `laravel-echo-server` 服务在大用户量下占用太多
的系统资源，于是就用 Go 重新造个轮子。

###  [USAGE](doc/USAGE.md)

### Features

+ [x] Websocket服务
+ [x] Redis订阅
+ [x] 定时输出数据统计
+ [x] 心跳检测
+ [x] HTTPS
+ [x] 数据接口
